
🛒 ShopEase Ecommerce Website
📌 Overview

ShopEase is a full-stack ecommerce web application built using React (Vite) for the frontend and Spring Boot for the backend. It allows users to browse products, manage cart, and complete checkout with a simple and responsive UI.

🚀 Features
🏠 Home page with product categories
🔐 User Login & Registration
🛍️ Product listing page
🛒 Add to Cart functionality
💳 Checkout page
⚙️ Admin dashboard
📱 Responsive UI design
🧰 Tech Stack
Frontend
React (Vite)
HTML, CSS, JavaScript
Backend
Spring Boot
Java
REST APIs
Database
MySQL / SQLite (based on configuration)

npm install
npm run dev
▶ Backend
cd Ecommerce_website/backend
./mvnw spring-boot:run

Backend runs at:

http://localhost:8081
